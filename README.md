## PROJECT DESCRIPTION:

Project Features:
- Local Authantication 
- Add Notes functionality
- Theme Changeing functionality ( Light & Dark Theme )
- Edit & Delete Notes functionality

Project Technologies:
- Dart language
- Flutter framework with Local Authantication
